1. Pendahuluan

Warung Kopi POS adalah aplikasi Point of Sale (POS) berbasis web yang dirancang khusus untuk membantu pengelolaan kedai kopi dan usaha kuliner sejenis. Aplikasi ini menyediakan solusi lengkap mulai dari pencatatan pesanan, pengelolaan menu, pengaturan meja, hingga laporan penjualan yang terperinci. Dengan tampilan yang modern dan mudah dipahami, aplikasi ini cocok digunakan oleh siapa saja tanpa memerlukan pelatihan khusus.

Aplikasi ini menggunakan sistem jual putus (sekali bayar), artinya Anda cukup membayar sekali untuk mendapatkan akses permanen selamanya. Tidak ada biaya langganan bulanan atau tahunan. Seluruh data penjualan Anda tersimpan secara lokal di perangkat masing-masing pengguna melalui localStorage browser, sehingga menjamin privasi dan keamanan data. Sistem autentikasi akun menggunakan Firebase Authentication, sementara kontrol aktivasi akun dikelola melalui Firebase Firestore. Arsitektur ini memastikan penggunaan bandwidth server yang sangat minimal, menjadikan aplikasi tetap responsif dan efisien.

Panduan ini akan membantu Anda memahami seluruh fitur yang tersedia, mulai dari cara mendaftar akun, proses aktivasi, hingga penggunaan setiap menu di dalam aplikasi. Bacalah panduan ini secara lengkap untuk memaksimalkan penggunaan aplikasi Warung Kopi POS di kedai kopi Anda.

2. Pendaftaran Akun Baru

Untuk mulai menggunakan aplikasi Warung Kopi POS, Anda perlu mendaftarkan akun terlebih dahulu. Proses pendaftaran sangat sederhana dan hanya memerlukan beberapa informasi dasar tentang kedai kopi Anda. Berikut adalah langkah-langkah lengkap untuk membuat akun baru:

Langkah Pendaftaran

  1. Buka aplikasi di browser melalui alamat yang telah disediakan oleh admin.
  2. Pada halaman utama, klik tombol Daftar yang terletak di bagian bawah form login.
  3. Isi formulir pendaftaran dengan data berikut: Nama Pemilik, Nama Toko, Alamat Email, Nomor Telepon, dan Password (minimal 6 karakter).
  4. Konfirmasi password dengan memasukkan ulang password yang sama.
  5. Klik tombol Daftar untuk mengirim data pendaftaran.
  6. Setelah berhasil, Anda akan menerima halaman Aktivasi Akun yang berisi kode aktivasi unik Anda.

Pastikan email yang Anda gunakan masih aktif dan dapat diakses, karena email tersebut akan menjadi identitas utama akun Anda untuk login dan komunikasi dengan admin. Gunakan password yang kuat dan mudah diingat.

Catatan Penting: Setelah mendaftar, akun Anda akan berstatus pending (menunggu). Anda belum dapat menggunakan aplikasi secara penuh sebelum akun diaktivasi oleh admin. Namun, Anda dapat mencoba aplikasi selama 24 jam menggunakan kode demo jika tersedia (lihat bagian Akun Demo).

3. Aktivasi Akun

Setelah berhasil mendaftar, langkah selanjutnya adalah mengaktifkan akun Anda. Terdapat dua cara untuk mengaktifkan akun: melalui aktivasi permanen (jual putus) atau melalui akun demo 24 jam. Kedua cara ini memiliki prosedur yang berbeda namun sama-sama mudah dilakukan.

3.1 Aktivasi Permanen (Jual Putus)

Aktivasi permanen memberikan Anda akses penuh selamanya dengan sekali pembayaran. Berikut langkah-langkahnya:

  1. Setelah mendaftar, Anda akan melihat halaman Aktivasi Akun yang menampilkan kode aktivasi dengan format WK-XXXXXX.
  2. Klik tombol Salin Kode untuk menyalin kode aktivasi ke clipboard, atau catat kode tersebut secara manual.
  3. Klik tombol Hubungi Admin via WhatsApp. Tombol ini akan membuka aplikasi WhatsApp dengan pesan otomatis yang sudah berisi data toko dan kode aktivasi Anda.
  4. Kirim pesan tersebut kepada admin dan tunggu balasan terkait informasi pembayaran.
  5. Setelah admin menerima pembayaran, akun Anda akan diaktifkan secara permanen. Klik tombol Cek Status Aktivasi atau login ulang untuk memasuki aplikasi.
Informasi: Kode aktivasi bersifat unik dan hanya dapat digunakan satu kali. Setelah diaktivasi, kode tersebut tidak dapat dipindahkan ke akun lain. Pastikan Anda menyimpan bukti pembayaran sebagai dokumen referensi.

3.2 Akun Demo (24 Jam)

Jika Anda ingin mencoba aplikasi sebelum membeli, Anda dapat menggunakan fitur akun demo yang berlaku selama 24 jam. Fitur ini memungkinkan Anda menjelajahi seluruh fitur aplikasi tanpa batasan:

  1. Pada halaman Aktivasi Akun, scroll ke bagian bawah hingga menemukan bagian Coba Demo 24 Jam.
  2. Masukkan kode demo dengan format DEMO-XXXXXX yang telah Anda terima dari admin.
  3. Klik tombol Mulai Demo untuk mengaktifkan mode demo.
  4. Aplikasi akan masuk ke mode demo selama 24 jam. Anda dapat melihat sisa waktu demo pada banner ungu di bagian atas aplikasi.
  5. Setelah 24 jam berakhir, Anda akan diminta untuk mengaktifkan akun secara permanen atau menghubungi admin.
Penting: Mode demo berfungsi penuh termasuk semua fitur pencatatan dan laporan. Semua data yang Anda masukkan selama masa demo akan tetap tersimpan setelah Anda beralih ke akun permanen, sehingga Anda tidak perlu memasukkan ulang data.

Tombol keluar (logout) juga terletak di bagian kanan atas header.

5. Menu Kasir (Pencatatan Pesanan)

Menu Kasir adalah halaman utama dan paling sering digunakan dalam aplikasi POS ini. Halaman ini menyediakan seluruh alat yang dibutuhkan untuk mencatat pesanan pelanggan, mengelola meja, dan melakukan proses pembayaran. Desain halaman ini dirancang agar kasir dapat bekerja dengan cepat dan efisien, bahkan saat jam sibuk.

5.1 Pengelolaan Meja

Bagian atas halaman Kasir menampilkan daftar seluruh meja dalam format grid. Setiap meja ditampilkan sebagai kartu yang menunjukkan nomor meja dan statusnya. Secara default, aplikasi menyediakan 30 meja yang dapat disesuaikan dengan jumlah meja aktual di kedai kopi Anda.

Untuk menyesuaikan jumlah meja, gunakan tombol tambah (+) dan kurang (-) yang terletak di samping label "Daftar Meja". Tombol ini memungkinkan Anda menambah atau mengurangi jumlah meja secara fleksibel sesuai kebutuhan. Jumlah meja yang telah diatur akan tersimpan otomatis, sehingga Anda tidak perlu mengaturnya ulang setiap kali membuka aplikasi.

Status Meja

Setiap meja memiliki dua status yang ditampilkan dengan warna berbeda. Meja kosong ditandai dengan warna putih standar, sedangkan meja yang sedang digunakan (memiliki pesanan aktif) akan ditandai dengan warna krem/coklat muda dan menampilkan ringkasan pesanan seperti jumlah item dan total harga. Hal ini memudahkan kasir untuk memantau kapasitas kedai secara sekilas.

5.2 Memilih Menu untuk Pesanan

Di bawah daftar meja terdapat katalog menu yang menampilkan semua item makanan dan minuman yang tersedia. Menu ini ditampilkan dalam format grid yang mudah dilihat, lengkap dengan nama menu, harga, dan ikon kategori. Terdapat fitur pencarian dan filter untuk memudahkan menemukan menu tertentu:

  • Pencarian: Ketik nama menu di kolom pencarian untuk menampilkan hasil yang sesuai secara real-time.
  • Filter Kategori: Gunakan tombol "Semua", "Minuman", atau "Makanan" untuk menampilkan menu berdasarkan kategori.
  • Menambah Pesanan: Ketuk/klik item menu untuk menambahkannya ke pesanan meja yang sedang aktif.

5.3 Proses Pemesanan

Untuk membuat pesanan baru, ikuti langkah-langkah berikut:

  1. Klik/tap kartu meja yang ingin digunakan. Meja kosong akan berubah menjadi "aktif" dan panel pesanan akan muncul.
  2. Pilih menu yang diinginkan dari katalog menu di bawah daftar meja. Setiap item yang diklik akan otomatis ditambahkan ke daftar pesanan.
  3. Jika diperlukan, atur jumlah item menggunakan tombol tambah (+) dan kurang (-) di samping setiap item.
  4. Setelah semua item dimasukkan, klik tombol Bayar untuk memproses pembayaran.
  5. Pada jendela pembayaran, pilih metode pembayaran (Tunai, QRIS, atau Transfer), masukkan jumlah diskon (jika ada), dan masukkan jumlah uang yang diterima untuk metode Tunai.
  6. Klik Konfirmasi Pembayaran untuk menyimpan transaksi dan mencetak struk.
  7. Setelah pembayaran berhasil, meja akan otomatis dikosongkan dan siap digunakan untuk pelanggan berikutnya.

5.4 Proses Pembayaran

Jendela pembayaran akan muncul ketika Anda mengklik tombol Bayar. Jendela ini menampilkan ringkasan pesanan lengkap beserta subtotal, opsi diskon dalam persentase, dan total akhir yang harus dibayar. Terdapat tiga metode pembayaran yang dapat dipilih:

6. Riwayat Penjualan

Menu Riwayat Penjualan menyediakan laporan terperinci mengenai seluruh transaksi yang telah dilakukan. Fitur ini sangat berguna untuk menganalisis performa penjualan, mengidentifikasi menu terlaris, dan membuat perencanaan bisnis yang lebih baik.

6.1 Laporan Harian

Laporan harian menampilkan seluruh transaksi yang terjadi pada tanggal tertentu. Secara default, aplikasi akan menampilkan data untuk hari ini. Anda dapat mengubah tanggal menggunakan date picker yang tersedia, lalu klik tombol "Cari" untuk memuat data. Laporan ini mencakup ringkasan jumlah transaksi, total pendapatan, dan rata-rata per transaksi. Selain itu, terdapat bagian "Menu Terlaris" yang menampilkan item-menu dengan penjualan terbanyak pada hari tersebut.

6.2 Laporan Bulanan

Laporan bulanan memberikan gambaran yang lebih luas tentang performa penjualan dalam satu bulan penuh. Pilih bulan yang ingin dilihat menggunakan month picker, lalu klik tombol "Cari". Laporan ini menampilkan ringkasan total pendapatan, jumlah transaksi, dan rata-rata harian selama satu bulan. Data ini sangat berguna untuk membandingkan performa antar bulan dan mengidentifikasi tren penjualan.

6.3 Laporan Kustom

Laporan kustom memungkinkan Anda menentukan rentang tanggal sendiri untuk analisis yang lebih fleksibel. Pilih tanggal mulai dan tanggal akhir, lalu klik "Cari". Fitur ini cocok digunakan untuk menganalisis penjualan selama periode tertentu seperti saat event khusus, hari libur, atau setelah promosi.

6.4 Export CSV dan Cetak

Setiap laporan dapat diekspor ke format CSV untuk keperluan analisis lebih lanjut di aplikasi spreadsheet seperti Microsoft Excel atau Google Sheets. Klik tombol Export CSV untuk mengunduh file. Selain itu, Anda juga dapat mencetak laporan langsung dari browser menggunakan tombol cetak (printer icon) yang tersedia. Fitur ini sangat membantu untuk dokumentasi atau pelaporan kepada pemilik usaha.

7. Kelola Menu

Menu Kelola memungkinkan Anda mengatur daftar makanan dan minuman yang tersedia di kedai kopi Anda. Anda dapat menambahkan menu baru, mengubah harga atau nama menu yang sudah ada, atau menghapus menu yang tidak lagi disajikan. Setiap perubahan akan langsung tercermin di halaman Kasir tanpa perlu me-restart aplikasi.

Menambah Menu Baru

  1. Buka menu Menu (ikon utensils) di navigasi bawah.
  2. Klik tombol Tambah di pojok kanan atas.
  3. Isi formulir: Nama Menu, Harga (dalam Rupiah), dan pilih Kategori (Minuman atau Makanan).
  4. Klik Simpan untuk menyimpan menu baru.

Mengedit atau Menghapus Menu

Pada daftar menu, setiap item memiliki tombol Ubah (ikon pensil) dan Hapus (ikon tempat sampah). Klik tombol Ubah untuk mengedit nama, harga, atau kategori menu. Klik tombol Hapus untuk menghapus menu dari daftar. Perhatikan bahwa menghapus menu tidak akan memengaruhi data transaksi yang sudah tercatat sebelumnya. Menu yang dihapus masih akan terlihat pada struk dan riwayat transaksi lama.

Tips: Secara default, aplikasi telah menyediakan 20 menu awal yang terdiri dari berbagai minuman kopi, minuman lainnya, dan makanan ringan. Anda dapat mengubah atau menambah menu sesuai dengan menu aktual di kedai kopi Anda.

9. Backup dan Restore Data

Fitur Backup dan Restore sangat penting untuk menjaga keamanan data Anda. Karena seluruh data penjualan disimpan secara lokal di perangkat (localStorage browser), ada risiko data hilang jika browser di-reset atau perangkat mengalami masalah. Fitur ini memungkinkan Anda membuat salinan cadangan dan memulihkan data kapan saja diperlukan.

Export Backup

Klik tombol Export Backup untuk mengunduh seluruh data Anda dalam format file JSON. File ini berisi data menu, transaksi, pengaturan toko, dan konfigurasi meja. Simpan file ini di lokasi yang aman seperti Google Drive atau flash disk. Disarankan untuk melakukan backup secara berkala, misalnya setiap akhir minggu, agar data Anda selalu terlindungi.

Import Restore

Jika Anda perlu memulihkan data (misalnya setelah ganti perangkat atau browser), klik tombol Import Restore dan pilih file backup JSON yang sebelumnya telah Anda simpan. Data yang ada akan digantikan dengan data dari file backup. Pastikan Anda tidak memiliki transaksi penting yang belum di-backup sebelum melakukan restore, karena data lama akan ditimpa.

Hapus Semua Data

Peringatan: Tombol Hapus Semua Data Lokal memungkinkan Anda menghapus seluruh data yang tersimpan di perangkat saat ini. Fitur ini berguna jika Anda ingin memulai dari awal atau membersihkan data percobaan. Perhatikan bahwa tindakan ini bersifat permanen dan tidak dapat dibatalkan, pastikan Anda telah membuat backup sebelum menghapus data.

10. Tips dan Pertanyaan Umum

Tips Penggunaan

  • Lakukan backup rutin: Buat salinan cadangan data setiap minggu untuk menghindari kehilangan data akibat masalah teknis pada browser atau perangkat.
  • Gunakan akun demo terlebih dahulu: Jika ragu, coba akun demo 24 jam untuk memastikan aplikasi sesuai dengan kebutuhan Anda sebelum membeli.
  • Atur jumlah meja sesuai kapasitas: Sesuaikan jumlah meja di pengaturan dengan jumlah meja aktual di kedai Anda agar tampilan lebih akurat.
  • Manfaatkan laporan penjualan: Gunakan fitur export CSV untuk menganalisis tren penjualan dan membuat keputusan bisnis yang lebih baik.
  • Perbarui menu secara berkala: Pastikan daftar menu selalu sesuai dengan menu yang tersedia agar pesanan tidak terjadi kesalahan.
  • Kelola password dengan baik: Gunakan password yang kuat dan tidak mudah ditebak. Jika lupa password, gunakan fitur "Lupa Password" di halaman login.

Pertanyaan Umum (FAQ)

1. Apakah data penjualan saya aman?
Ya. Seluruh data penjualan disimpan secara lokal di perangkat Anda (localStorage browser) dan tidak dikirim ke server manapun. Data hanya ada di perangkat Anda. Namun, karena data tersimpan di browser, pastikan Anda rutin melakukan backup menggunakan fitur Export Backup.
2. Bagaimana jika saya ganti perangkat atau browser?
Anda perlu melakukan Export Backup di perangkat lama, kemudian Import Restore di perangkat baru. File backup berisi seluruh data Anda dan dapat dipindahkan ke perangkat mana pun.
3. Apakah aplikasi bisa digunakan tanpa koneksi internet?
Sebagian besar fitur dapat digunakan offline karena data disimpan lokal. Namun, untuk login dan pengecekan status akun, diperlukan koneksi internet. Setelah berhasil login, Anda dapat mengoperasikan aplikasi secara offline untuk transaksi sehari-hari.
4. Bagaimana cara menghubungi admin?
Anda dapat menghubungi admin melalui tombol WhatsApp yang tersedia di halaman Aktivasi Akun atau langsung ke nomor WhatsApp admin yang tertera.
5. Apakah ada biaya langganan bulanan?
Tidak. Aplikasi ini menggunakan sistem jual putus (sekali bayar). Anda hanya perlu membayar sekali untuk mendapatkan akses permanen selamanya. Tidak ada biaya tersembunyi atau langganan.
6. Berapa jumlah maksimal meja yang didukung?
Aplikasi mendukung hingga 100 meja. Anda dapat menyesuaikan jumlah meja menggunakan tombol tambah dan kurang di halaman Kasir.
7. Bagaimana jika saya lupa password?
Klik link "Lupa Password?" di halaman login, masukkan email terdaftar, dan ikuti instruksi di email yang dikirimkan oleh Firebase untuk mereset password Anda.
