Cara Setup Google Drive Backup

1

Buka Google Cloud Console

Kunjungi console.cloud.google.com dan login dengan akun Google yang sama.

2

Buat Project Baru atau Pilih yang Ada

Buka project Firebase Anda (warkop-app-20449) atau buat baru.

3

Aktifkan Google Drive API

Menu: APIs & Services → Library → cari "Google Drive API" → klik Enable.

4

Buat OAuth 2.0 Client ID

Menu: APIs & Services → Credentials → Create Credentials → OAuth client ID
Type: Web application
Nama: WARKOPOS Backup
Authorized JavaScript origins: tambahkan URL website Anda (misal: https://yourdomain.com)

Klik Create → Copy Client ID yang muncul.

5

Konfigurasi Consent Screen

Menu: APIs & Services → OAuth consent screen
User Type: External
Isi App name, User support email, Developer contact email
Scope yang dibutuhkan: .../auth/drive.appdata
Publish app (test mode cukup untuk pemakaian sendiri).

6

Paste Client ID di Settings

Paste Client ID yang sudah di-copy ke kolom "Google OAuth Client ID" di halaman Settings ini, lalu klik Simpan.

Penting

Pastikan URL website Anda ditambahkan di "Authorized JavaScript origins" pada OAuth Client ID, atau Google Drive login tidak akan berfungsi.
